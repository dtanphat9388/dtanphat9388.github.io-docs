# Prerequisite

  • Trước khi bắt đầu, mình cần cài đặt 1 số thứ cần thiết để phục vụ cho việc lập trình

# IDE

# VSCode

  • Settings Sync: backup VSCode settings đến gist bất cứ khi nào có thay đôi
  • GitLens: Tiện ích quản lý code cho git
  • Git Project Manager: Tiện ích quản lý code cho git
  • Ayu: theme đẹp, có light và dark
  • css-triggers: cho biết tính chất của từng thuộc tính để tối ưu
  • Quokka.js: giúp kiểm tra lỗi trong khi viết code
  • Mocha sidebar: chạy các test trực tiếp trong VSCode
  • Debugger for Chrome: add Chrome Debug into VSCode

Intergate VSCode to terminal

  • VSCode: press Command+Shift+P and type shell command
  • Select Shell Command: Install 'code' command in PATH

After then, in terminal you can type code<file> to edit file by VSCode

# Software

  1. NodeJS: giúp chạy JS trên localhost (bên ngoài môi trường browser)
  2. git-scm: sử dụng các git command để thao tác với git server
